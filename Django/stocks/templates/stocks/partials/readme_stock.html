Technology Stack

  • 🐬 MySQL – Stores all stock data securely. Includes creation and configuration of the database and its user, integration with the web scraper, and seamless connection to Django for data management.
  • 🕷️ Selenium – Automatically scrapes live stock data, including detailed stock information and company profiles, ensuring the database is always up-to-date.
  • ֎ ChatGPT API – Provides AI-driven insights and analysis. Used to fill in missing information and enhance the dataset with predictive or descriptive insights.
  • 🐍 Python ETL – Efficiently extracts, transforms, and loads data into the database, ensuring consistency and accuracy across all records.
  • 🌐 Django, HTML, CSS, JavaScript – Powers an interactive and responsive web interface. Designed as a segmented web application for clear visualization and easy navigation of stock data.
  • 📦 GitHub + venv – Version control for project management and a virtual environment for isolated, reproducible Python dependencies.
  • 🐧 Linux server – Provides a stable and secure deployment environment to run the Django server and manage background ETL processes.
  • 📦 Future: Docker – Planned containerization for easy deployment, scaling, and portability of the application.
